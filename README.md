# Dashboard

Este projeto é um dashboard interativo desenvolvido em React para visualização de dados de faturamento e vendas, utilizando gráficos e tabelas dinâmicas. O sistema permite exportar o dashboard em formato PNG e PDF, facilitando o compartilhamento e análise dos dados.

Tecnologias utilizadas:
React
Recharts (gráficos)
@tanstack/react-table (tabelas dinâmicas)
html2canvas (exportação para imagem)
jsPDF (exportação para PDF)
JavaScript
CSS
